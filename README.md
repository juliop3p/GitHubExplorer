## Github Explorer feito em Angular

#### Replicado através da aplicação feita pela Rocketseat

### Setup do projeto

**Requisitos**

- Node.js
- Angular CLI

1. Passo: Clonar o projeto.

2. Passo: Instalar as dependências.

   `npm install`

3. Passo: Startar o projeto.
   `ng serve`
